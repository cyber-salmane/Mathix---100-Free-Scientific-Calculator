# Mathix — AI Math Solver & Scientific Calculator

Mathix is a free, fast, and powerful **AI-powered math platform** designed for students, engineers, and anyone who works with mathematics.

## 🚀 Live Demo
👉 https://mathix.netlify.app/

---

## ✨ Features

- 🤖 AI Math Solver (ask anything, get step-by-step solutions)
- 📸 Math Scanner (scan handwritten or printed problems and solve them instantly)
- 📊 Graphing Calculator (visualize functions in real-time)
- 🧮 Scientific Calculator (advanced math functions)
- ➗ Equation Solver (solve, factor, simplify, derivatives, and more)
- 🔄 Unit Converter
- 📚 Step-by-step explanations (Easy / Medium / Detailed modes)
- 🔗 Share results instantly with a link
- ⚡ 100% free — no login, no ads
